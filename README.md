# AI Smart Helmet

AI based smart helmet system for accident detection, drowsiness detection and emergency alert using Raspberry Pi and ESP32.

## Features

- Crash detection using MPU6050 IMU
- Helmet wear detection using touch sensor
- Emergency SMS alert via GSM module
- 10 second alarm cancel window
- GPS location link in SMS

## Hardware Used

- Raspberry Pi Zero 2W
- ESP32
- MPU6050 IMU
- SIM800L GSM module
- Touch sensor
- Buzzer
- Push button

## Project Status

Work in progress. GPS UART integration and RTC module will be added in future versions.
